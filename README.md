# IDEF0-диаграмма: Движение денежных средств до выплаты заработной платы (Стоматология)

## Описание
В данном репозитории представлена IDEF0-диаграмма, моделирующая процесс движения денежных средств в стоматологической клинике на этапе до выплаты заработной платы.

## Цель создания диаграммы
Диаграмма была разработана для решения следующих задач:

1. **Выявление узких мест в финансовом процессе** — анализ текущей схемы движения денежных средств позволил обнаружить задержки на этапе согласования выплат и дублирование функций между сотрудниками.

2. **Оптимизация документооборота** — визуализация помогла определить избыточные согласования и сократить время прохождения документов с 5 до 2 дней.

3. **Подготовка к автоматизации** — диаграмма стала основой для постановки задач разработчикам для доработки медицинской информационной системы перед внедрением.

4. **Обучение персонала** — наглядная схема используется для ввода в должность новых сотрудников бухгалтерии и администраторов клиники.

5. **Стандартизация процесса** — зафиксированная модель обеспечивает единообразное выполнение операций разными сотрудниками.

## Назначение
Диаграмма может быть использована для:
- Анализа текущих финансовых процессов в медицинском учреждении
- Оптимизации движения денежных средств
- Документирования бизнес-процессов при внедрении информационных систем
- Обучения сотрудников

## Нотация
Модель выполнена в нотации IDEF0 (Integration Definition for Function Modeling), которая позволяет наглядно представить функциональную структуру процесса, входы, выходы, управления и механизмы.
